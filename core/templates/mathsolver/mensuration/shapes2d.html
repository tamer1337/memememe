Plane Figures: Area and Perimeter

Shape Area Perimeter Diagram
Square A = a² P = 4a
Rectangle A = l × w P = 2(l + w)
Circle A = πr² P = 2πr
Equilateral Triangle A = (√3/4)a² P = 3a
Isosceles Triangle A = (b/4)√(4a² - b²) P = 2a + b
Scalene Triangle A = √[s(s-a)(s-b)(s-c)]
s = (a+b+c)/2		 P = a + b + c
Right Triangle A = (ab)/2 P = a + b + √(a² + b²)
Rhombus A = (d₁ × d₂)/2 P = 4a
Parallelogram A = b × h P = 2(a + b)
Trapezoid A = h(a + b)/2 P = a + b + c + d

Legend:

  • a, b, c, d = side lengths
  • l, w = length, width
  • r = radius
  • h = height
  • d₁, d₂ = diagonals (rhombus)
  • s = semi-perimeter (Heron's formula)
