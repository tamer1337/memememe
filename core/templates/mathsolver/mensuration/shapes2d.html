Плоские фигуры: площадь и периметр

Фигура Площадь Периметр Схема
Квадрат S = a² P = 4a
Прямоугольник S = l × w P = 2(l + w)
Круг S = πr² P = 2πr
Равносторонний треугольник S = (√3/4)a² P = 3a
Равнобедренный треугольник S = (b/4)√(4a² - b²) P = 2a + b
Разносторонний треугольник S = √[s(s-a)(s-b)(s-c)]
s = (a+b+c)/2		 P = a + b + c
Прямоугольный треугольник S = (ab)/2 P = a + b + √(a² + b²)
Ромб S = (d₁ × d₂)/2 P = 4a
Параллелограмм S = b × h P = 2(a + b)
Трапеция S = h(a + b)/2 P = a + b + c + d

Обозначения:

  • a, b, c, d — длины сторон
  • l, w — длина и ширина
  • r — радиус
  • h — высота
  • d₁, d₂ — диагонали (у ромба)
  • s — полупериметр (формула Герона)
